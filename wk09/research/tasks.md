# Evaluation Tasks - Week 9

## Task 1: Add new task

**Scenario**:
You are going shopping tomorrow and realise you have forgotten to buy your friend a birthday card

**Instructions**:
1. Add new task: "Buy birthday card"
2. Ensure it is added to the list

**Success criteria**:
- Adding a task woks
- Immediately updates the list

**Inclusion focus**:
- Keyboard only for adding the task
- Ensure error message for blank title

---

## Task 2: Edit task

**Scenario**:
You realised you mistyped the due date

**Instructions**:
1. Find the task
2. Use edit button to change the due date
3. Ensure edit is saved

**Success criteria**:
- Edit button works
- Saves the change immediately

**Inclusion focus**:
- Works successfully for a screen reader

---

## Task 3: Delete a task

**Scenario**:
You have completed a task and wish to remove it from the list

**Instructions**:
1. Find the task
2. Use delete button to delete it
3. Ensure the task is removed

**Success criteria**:
- Delete button works
- Deletes the task immediately

**Inclusion focus**:
- Makes you confirm deleting the task before you do it

---

## Task 4: Find a task

**Scenario**:
You can't remember when your assigment is due so check your to-do list

**Instructions**:
1. Use search feature
2. Start typing the saved task to find it
3. Ensure you can see the task and find out the due date
**Success criteria**:
- Allows you to type
- Starts showing you matches in a list form
- Results output as you are typing, no delay

**Inclusion focus**:
- Works successfully for a screen reader

---

## Metrics per Task

For each task, record:
- **Time-on-task** (seconds from start to completion)
- **Success** (1 = completed, 0 = abandoned)
- **Error count** (validation errors, wrong clicks)
- **Mode** (HTMX or no-JS)
